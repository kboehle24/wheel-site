# Wheel for Winner selection

A clean GitHub Pages-ready spinning wheel website with 12 preloaded images, a trigger button, winner popup, confetti, and celebration music after the spin.

## Files
- `index.html`
- `styles.css`
- `script.js`
- `assets/` with all images

## Upload to GitHub
1. Create a new public GitHub repository.
2. Upload all files and folders from this project.
3. In the repository, open **Settings > Pages**.
4. Under **Source**, choose **Deploy from a branch**.
5. Select **main** and **/(root)**, then save.
6. Wait a few seconds and open the GitHub Pages link.
